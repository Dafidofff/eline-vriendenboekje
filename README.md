# Eline's Promotieboekje

A colourful, interactive friend booklet for Eline's PhD defence.

## Getting started

1. **Add your background image** — place your image file as `background.png` in the root of this repository. The site will use it as the full-page background.
2. Open `index.html` in a browser (or deploy via GitHub Pages) and you're good to go!

## Structure

```
/
├── index.html       # The complete single-page frontend
└── background.png   # Your background image (add this yourself!)
```

## Features (frontend)

- Fun PhD-themed questions in Dutch
- Photo upload with live preview (drag & drop supported)
- Print / Save as PDF button
- Colourful responsive design
- Backend integration point ready — form `submit` handler in `index.html`
